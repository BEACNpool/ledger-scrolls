Ledger Scrolls

The protocol on one page · v1 · July 2026
This document is itself stored on Cardano as a scroll.

Ledger Scrolls puts files inside the Cardano blockchain — not a link, not a hash of a file kept somewhere else; the actual bytes, in transactions, replicated by thousands of independent nodes. Once written, a scroll cannot be edited, cannot be deleted, and is timestamped by the block that sealed it. Anyone can read it, free, forever. It costs a few dollars — once. No subscription, no pinning service, no custodian.

How it works

POINTER MANIFEST PAGES YOUR FILE one tx id ▶ locked on-chain ▶ plain txs ▶ concat → gunzip (share it forever: type, carrying the → SHA-256 ✓ as a QR) hashes, page list actual bytes verified, rendered

A pointer (one transaction id) leads to a manifest locked in an unspendable UTxO — an always-fail script address every node carries forever. The manifest declares the content type, lists the exact transactions holding the file's pages, and commits to the SHA-256 of the finished file. A reader fetches, reassembles, and proves the bytes before showing them. The golden rule: a viewer never shows you bytes it could not verify.

Verify with your own hands

The whole protocol is two public queries and arithmetic — free, keyless, ~30 lines in any language:

# 1 — the manifest (type, hashes, page list): curl -X POST https://api.koios.rest/api/v1/utxo_info \ -d '{"_utxo_refs":["<manifest-tx>#0"],"_extended":true}' -H 'Content-Type: application/json' # 2 — the pages it names (label 22025 holds the payload): curl -X POST https://api.koios.rest/api/v1/tx_metadata \ -d '{"_tx_hashes":["<page-tx-1>","..."]}' -H 'Content-Type: application/json' # concatenate segments → gunzip → sha256sum — must equal the manifest's hash

The five rules

  • Permanent means permanent. No edit, no delete — never inscribe secrets.
  • Rehearse on testnet. Same bytes, free mistakes.
  • Hash before, verify after. Reconstruct from chain before announcing.
  • Keep receipts. Tx hashes are how civilization finds it again.
  • Self-contained files only. Anything that phones home is a broken promise.
