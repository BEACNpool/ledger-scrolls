How it works

Your signature is a transaction

Signing sends the smallest output Cardano allows (~0.86 ADA, the protocol's min-UTxO) to the well-known always-fail script address — an address with no spending key. That anchor is what lets this page find your signature with no server; your name and message travel in the transaction's metadata (label 22031). A transaction can never be free — the ~0.17 network fee goes to the protocol, and nobody else gets anything.

The book is the blockchain

This page holds no database. Every time it opens, it asks the chain for the signature transactions and lays out the pages fresh. Mirror this file anywhere — the book follows, because the book is the chain.

Proof of authorship

Every entry was signed by its author's own wallet keys — attribution is cryptographic, not claimed. The small print under each signature links straight to the transaction.
