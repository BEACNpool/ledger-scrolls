How a Ledger Scroll works

Your file is written byte-for-byte into Cardano transactions and sealed by a manifest that no one — including you — can ever spend, edit, or delete. Readers rebuild the file from the chain and verify its SHA-256 fingerprint before a single byte is shown.

Pages 12,160 bytes per transaction
The file is split into pages riding in ordinary transactions. No tokens, nothing locked per page — just bytes in the ledger's permanent history.
The manifest one small locked coin, forever
Lists every page and seals the file's fingerprint. It sits at an address that can never be spent, so every Cardano node carries it, always. Tiny files skip the pages entirely — the whole file fits inside this one locked coin.
Zero tokens nothing to custody, move, or lose
The very first scrolls (2026) carried pages in NFTs — still perfectly readable, but ~6× the cost. Today's format needs no tokens at all.
Assumptions & formulas (tap to open)

Estimates use Cardano mainnet parameters (2026): fee = 0.155381 ADA + 0.000044 ADA per byte of transaction; a locked coin carries ≈ 0.00431 ADA per byte of datum. Pages hold 12,160 bytes each (ceil(encoded ÷ 12,160) transactions, plus one manifest per 400 pages). Files that fit in a single datum (≤ ~14 KB) are priced as one locked coin when that's cheaper. Real costs vary a few percent with wallet layout — the mint tools print the exact price before you sign. Gzip here is your browser's; the mint pipeline uses deterministic gzip (gzip -n -9), which may differ by a few bytes.

Reference points from live scrolls: the on-chain tutorial (18 KB HTML → 7.4 KB gzip) cost ~0.9 ADA in fees + ~1.6 ADA locked. Full protocol details: the specs.

⚡ Mint it right now — no node, no command line

This page can write your file to Cardano directly from your browser wallet (Eternl, Lace, Typhon, Vespr…). The transaction is built locally, your wallet shows you exactly what it spends before you approve, and nothing — not the file, not a key — ever leaves your machine except the signed transaction itself.

Never used Cardano? Start here (10 minutes, three steps)
  1. Get a wallet. Install Eternl (desktop browser extension or mobile app). Write the seed phrase on paper — it is the wallet.
  2. Fund it. Buy ADA on any exchange and withdraw to your wallet's receive address. Drop your file above first — the price it shows plus a couple of ADA for fees is your funding target.
  3. Come back here in the wallet's browser. Desktop: the same browser as the extension. Mobile: open this page inside the wallet app's built-in dApp browser. Then drop your file and follow the buttons — the page does the rest and proves the result back to you from the chain.

Plain truth about privacy: a scroll is forever and unsigned by default, but the transaction that minted it traces to the wallet that paid — and exchange accounts are identity-verified. If who minted it must stay secret, have someone you trust mint it for you: the SHA-256 proves your bytes arrived untouched no matter whose wallet paid the fee.
