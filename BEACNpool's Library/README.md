# BEACNpool's Library

Every scroll BEACNpool has written into the Cardano blockchain — **20 live
on mainnet**, permanently readable and verifiable by anyone, forever.

This is the human-browsable catalog: one entry per scroll, with every key you
need to find and verify it — the pointer, the channel policy and asset (where
it belongs to a publisher channel), the content details, and the SHA-256
fingerprint checked on every read.

- **Read any of them now:** [The Main Viewer](https://beacnpool.github.io/ledger-scrolls/)
  — paste a pointer, or click a scroll's "Open it" link below.
- **What the keys mean:** [pointer vs policy ID vs tx hash](../docs/FINDING_SCROLLS.md).
- **Machine-readable:** [card-catalog.csv](../examples/card-catalog/card-catalog.csv)
  (itself a scroll on-chain) · full technical record: [docs/SCROLL_INVENTORY.md](../docs/SCROLL_INVENTORY.md).

Every pointer, policy ID, and hash below is public and checkable on any
Cardano explorer. Nothing here is hosted — the bytes live on the chain.
